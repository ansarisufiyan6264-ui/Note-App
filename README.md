# 📝 Notes App

A simple Notes App built with HTML, CSS, and JavaScript.
It allows users to add, edit, and delete notes.
All notes are stored in LocalStorage, so they remain available even after refreshing the page.

## 🚀 Features

- Add Note
- Edit Note
- Delete Note
- LocalStorage Support
- Data Persists After Refresh

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript
- LocalStorage
